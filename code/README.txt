This is XMerge v24.12.1

Changes in v24.12.1

- Added a tab-delimited (.txt) output option. This helps when working 
  with those AutoCAD ATTOUT files.
- Improved error-catching when using command-line options, particularly -u.
- Unattended mode does not modify target LastXMerge.ini files in any way.
- Updated the Help file to include '-t'.
- General cleaning.

Changes in v24.09.1

- changed to YY.MM.iter method of numbering versions. 
- the displayed Export (output) filename does not get or need an extension.
- resolved output file extensions based on checkboxes.
- user selections are now saved in LastXMerge.ini as they should be.
- updated Help file to include Unattended Mode info.
